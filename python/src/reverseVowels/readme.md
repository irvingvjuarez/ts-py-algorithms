# Invertir vocales

Dada una cadena de caracteres, invierte solo todas las vocales de la cadena.
Las vocales son ‘a’, ‘e’, ‘i’, ‘o’ y ‘u’, ‘A’, ‘E’, ‘I’, ‘O’, ‘U’.

### Ejemplo 1

```py
# Entrada:
s = "hola"

# Salida:
"halo"
```

### Ejemplo 2

```py
# Entrada:
s = s = "leetcode"

# Salida:
"leotcede"
```
